# 聚合类

[聚合体](https://zh.cppreference.com/w/cpp/language/aggregate_initialization)是下列类型之一：

1. 数组类型
2. 符合以下条件的类(struct/class)类型:

· 没有用户声明的构造函数(>= C++11)
· 没有用户提供、继承或explicit的构造函数(>= C++11, < C++20)
· 没有用户声明或继承的构造函数(>=C++20)
· 没有私有或受保护的直接(C++17起)非静态数据成员
· 没有基类(< C++17)
· 没有虚基类(>= C++17)
· 没有私有或受保护的直接基类(>= C++17)
· 没有虚成员函数
· 没有默认成员初始化器(>= C++11, < C++14)
